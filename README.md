## System requirements

- CMake
- Python 2.7
- C++14 Compiler

### macOS

```
$ brew install cmake
```

### Ubuntu 16.04 TLS

```
$ sudo apt-get update
$ sudo apt-get install cmake g++ python libfontconfig1-dev \
    libx11-dev libxcomposite-dev libgl1-mesa-dev libglu1-mesa-dev freeglut3-dev -y
```

## How to build

```
$ cmake .
$ make
```
